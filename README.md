# User Profile

A simple Python console application that collects user information and stores it in a dictionary.

## Features

- Asks the user to enter their name
- Asks the user to enter their age
- Asks the user to enter their city
- Stores the entered data in a Python dictionary
- Displays the user's profile
- Checks if the city is stored in the profile

## Technologies

- Python 3
- Python dictionaries
- User input
- Conditional statements
- F-strings

## Purpose

This project was created to practice working with Python dictionaries, user input, and basic data processing.
